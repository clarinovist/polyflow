## Prinsip Strategis

PolyFlow tidak diposisikan sebagai ERP umum. PolyFlow diposisikan sebagai **ERP operasional untuk pabrik plastik converting Indonesia** yang membantu owner, produksi, gudang, dan finance mengontrol:

- stok bahan baku dan barang jadi,
- proses mixing, extrusion, packing,
- scrap/reject,
- HPP dan margin,
- maklon dan bahan milik customer.

## Urutan Eksekusi yang Disarankan

1. Validasi positioning ke 5-10 calon customer.
2. Jalankan beta cohort kecil, jangan full rollout sekaligus.
3. Pakai paket `Stock Control` atau `Production Control` sebagai entry point.
4. Baru upsell ke costing, finance, dan maklon setelah data operasional mulai disiplin.
